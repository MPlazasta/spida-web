# SPIDA Web

Site vitrine de l'application **SPIDA** — GPS qui calcule des itinéraires évitant les ralentisseurs, pensé pour les véhicules bas, motos et scooters.

## Stack

Site statique en HTML/CSS, sans framework.

- `index.html` — page d'accueil (10 sections : hero, produit, modes, navigation, stats, communauté, témoignages, à propos, footer)
- `styles.css` — styles complets + responsive

## Développement

Ouvrir `index.html` directement dans un navigateur, ou servir le dossier :

```bash
python -m http.server 8000
```
